📋 PRÉ-REQUIS
─────────────────────────────────────────────────────────────────────────────
• Node.js (version 18 ou supérieure)
• npm (installé avec Node.js)


🚀 LANCEMENT RAPIDE
─────────────────────────────────────────────────────────────────────────────

1. Ouvrir un terminal dans le dossier du projet :
   cd "C:\Users\Gabriel\Documents\projectgandw\DDoS Idle Game"

2. Installer les dépendances (première fois uniquement) :
   npm install

3. Lancer le jeu (frontend + backend ensemble) :
   npm run dev

4. Ouvrir votre navigateur à l'adresse :
   http://localhost:5173


📝 COMMANDES DISPONIBLES
─────────────────────────────────────────────────────────────────────────────

• npm run dev
  → Lance le frontend ET le backend simultanément (recommandé)

• npm run dev:client
  → Lance uniquement le frontend (Vite) sur http://localhost:5173

• npm run dev:server
  → Lance uniquement le backend (Node.js) sur le port 3000

• npm run build
  → Compile le projet pour la production


⚠️ EN CAS DE PROBLÈME
─────────────────────────────────────────────────────────────────────────────

ERREUR : "Port already in use" (port déjà utilisé)
─────────────────────────────────────────────────────────────────────────────
Solution Windows :
  taskkill /F /IM node.exe
  Puis relancer : npm run dev

Solution Linux/Mac :
  killall node
  Puis relancer : npm run dev


ERREUR : Backend non connecté
─────────────────────────────────────────────────────────────────────────────
1. Vérifier que le backend tourne bien sur le port 3000
2. Ouvrir la console du navigateur (F12) pour voir les erreurs
3. Vérifier que les deux serveurs sont lancés avec : npm run dev


ERREUR : Module manquant
─────────────────────────────────────────────────────────────────────────────
  npm install
  Puis relancer : npm run dev


🎮 UTILISATION DU MODE DÉMO
─────────────────────────────────────────────────────────────────────────────
• Activer "Demo Mode" dans le panneau Attack Configuration
• Le jeu fonctionne normalement SANS envoyer de vraies requêtes
• Utile pour tester sans impact sur les serveurs cibles


⚡ ARRÊTER LE JEU
─────────────────────────────────────────────────────────────────────────────
Dans le terminal : Ctrl + C (deux fois si nécessaire)


📊 PORTS UTILISÉS
─────────────────────────────────────────────────────────────────────────────
• Frontend (Vite)  : http://localhost:5173
• Backend (API)    : http://localhost:3000
• Socket.io        : ws://localhost:3000
